# Arquitectura Back-meli

## Stack
* ReactJS
* Material UI
* Styled Components
* 
## Para iniciar la aplicación en modo desarrollo

1 - Instalar depedencias
```
$ npm install
```

2 - Levantar entorno
```
$ npm run start
```

## Para Implementar la aplicación en modo productivo

1 - Instalar depedencias
```
$ npm install
```

2 - Buildear solución
```
$ npm run build
```

3 - Levantar entorno 
```
$ npm run start
```
